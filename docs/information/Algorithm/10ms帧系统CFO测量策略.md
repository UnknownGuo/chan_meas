# 10ms 帧系统 CFO 测量策略

> 本文讨论在硬件只能输出 10ms 一帧的约束下，如何通过调整帧内平均数 $M$ 来改善乃至估计真实 CFO。
> 结合内容：[[真实CFO估计方案（需连续获得15帧不平均ZC）]]、[[B2B频偏估计与OTA校准方案]]

---


## 2. 减少平均数（M = 2 或 3 替代 M = 15）

### 2.1 两个相互竞争的效应

| 效应 | 平均数少（M 小）| 平均数多（M 大）|
|------|---------------|---------------|
| CFO 幅度损失 | 小（有利）      | 大（不利）      |
| 噪声抑制     | 弱（不利）      | 强（有利）      |

**综合有效 SNR**（正比于 $g^2(M,\Delta\phi) \times M$）：

$f_{CFO} = 7\ \text{kHz}$ 时：

| $M$ | $g^2$ | 有效 SNR 因子 | 相比 $M=2$ |
|-----|-------|-------------|-----------|
| 2   | 0.951 | 1.90        | 0 dB      |
| 3   | 0.874 | 2.62        | +1.4 dB   |
| 5   | 0.648 | 3.24        | +2.3 dB   |
| 15  | 0.012 | 0.18        | **−10.2 dB** |

$f_{CFO} = 1\ \text{kHz}$ 时（小频偏），各 $M$ 差异极小，少平均数没有明显优势。

### 2.2 相位精度

减少 $M$ 后，每帧相位测量的噪声更低（幅度更大），帧间相位差 $\delta\phi[k]$ 的方差更小，CFO（余数）估计更干净。这正是 B2B 数据中 OTA（多径噪声大）比不过线缆直连测量的原因。

### 2.3 根本限制不变

减少 $M$ 只影响帧内，**帧间时间间隔 $T_{frame} = 10\ \text{ms}$ 不变**，Nyquist 极限仍然是 ±50 Hz。用 $M=2$ 测量得到的仍然是余数 CFO。

### 2.4 建议

- 当怀疑 CFO 较大（信号 SNR 明显低于预期）时，换用 $M = 2$ 或 $M = 3$ 录制一段 B2B 数据；
- 此时帧间相位斜率估计更准确，线性拟合结果更可信；
- 但仍需配合 [[B2B频偏估计与OTA校准方案]] 解释余数 CFO 的含义。

## 3. 顺序多 $M$ 测量（M = 2, 3, 4, 5）

### 3.1 相位角度：无任何改善

**结论先行**：顺序用不同 $M$ 测量，帧间相位差所携带的 CFO 信息完全相同，与 $M$ 的选择无关。

**推导**：$M$ 个 ZC 相干平均后，第 $k$ 帧的复数相位为：

$$\Phi_k^{(M)} = \phi_0^{(M)} + 2\pi f_{CFO} \cdot k T_{frame} + \underbrace{2\pi f_{CFO} \cdot \frac{M-1}{2} T_{ZC}}_{\text{帧内固定偏置}}$$

对帧间差分：

$$\delta\Phi^{(M)} = \Phi_k^{(M)} - \Phi_{k-1}^{(M)} = 2\pi f_{CFO} \cdot T_{frame}$$

**帧内固定偏置在差分中完全消去**，因此无论 $M = 2, 3, 4, 5$，帧间相位差都只包含 $2\pi f_{CFO} T_{frame}$ 这一项。不同 $M$ 的测量给出完全相同的余数 CFO 信息，多次顺序测量无法通过相位扩展不模糊范围。

### 3.2 幅度角度：可以粗估真实 CFO

每次 $M$ 值测量的主径功率均值（排除噪声）：

$$\bar{P}^{(M)} = P_0 \cdot g^2(M,\, \Delta\phi) = P_0 \cdot \left[\frac{\sin(M\pi f_{CFO} T_{ZC})}{M\,\sin(\pi f_{CFO} T_{ZC})}\right]^2$$

其中 $P_0$ 是无频偏时的真实功率，$\Delta\phi = 2\pi f_{CFO} T_{ZC}$。

**消去未知量 $P_0$**，取不同 $M$ 的功率比值：

$$\rho(M_1, M_2) \triangleq \frac{\bar{P}^{(M_2)}}{\bar{P}^{(M_1)}} = \frac{M_1^2 \cdot \sin^2(M_2 \pi f_{CFO} T_{ZC})}{M_2^2 \cdot \sin^2(M_1 \pi f_{CFO} T_{ZC})}$$

只有一个未知数 $f_{CFO}$（或等价地 $\Delta\phi$），$M = 2, 3, 4, 5$ 给出 $\binom{4}{2} = 6$ 个方程，超定系统，可用最小二乘拟合求解。

**方法步骤**：

1. 在 B2B 条件下依次录制 $M = 2, 3, 4, 5$ 的数据，每次至少 1 分钟；
2. 对每次录制的每帧，取主径 bin（非相干平均 PDP 的峰值位置）功率，取均值得到 $\hat{P}^{(M)}$；
3. 计算所有 $\binom{4}{2} = 6$ 组功率比 $\hat{\rho}(M_i, M_j)$；
4. 对 $\Delta\phi \in [0,\ \pi]$ 做网格搜索（或 scipy 非线性拟合），使理论比值与测量比值的残差平方和最小；
5. 还原真实 CFO：$\hat{f}_{CFO} = \hat{\Delta\phi} / (2\pi T_{ZC})$。

### 3.3 精度与适用范围

幅度比的敏感度随 $f_{CFO}$ 变化很大：

| $f_{CFO}$ | $\Delta\phi$ (rad) | $\rho(2,5) - 1$ | 可分辨性 |
|-----------|-------------------|-----------------|---------|
| 1 kHz     | 0.064             | 1.1%            | 需极高 SNR，难 |
| 3 kHz     | 0.192             | 9.5%            | 中等 SNR 可行 |
| 7 kHz     | 0.451             | 46.6%           | 容易     |
| 10 kHz    | 0.644             | 110%            | 非常明显  |

**结论**：

- $f_{CFO} \gtrsim 3\ \text{kHz}$ 时，幅度法有效，可以粗估真实 CFO（误差量级约数百 Hz）；
- $f_{CFO} \lesssim 1\ \text{kHz}$ 时，各 $M$ 之间幅度差异不足 2%，在噪声中无法分辨；
- 幅度法精度远低于新 1s 软件的相位法（单 burst 精度约 66 Hz，5 分钟约 4 Hz）。

### 3.4 注意事项

- 四次测量必须在**相同物理条件**（相同 B2B 链路、相同温度）下进行，否则 $P_0$ 的变化会干扰功率比；
- 主径 bin 可能在不同 $M$ 下略有不同（不同帧内时延解析度），需要手动对齐或取同一 bin；
- 应在噪声估计后扣除热噪声项 $\sigma_n^2 / M$，否则高 $M$ 的功率会被系统性低估。

## 4. 三种方案综合对比

| 维度        | 原方案 M=15     | 改为 M=2/3 | 顺序多M（幅度法）                      | 新 1s 软件（相位法）           |
| --------- | ------------ | -------- | ------------------------------ | ---------------------- |
| 帧率 / 输出间隔 | 10ms         | 10ms     | 10ms                           | 1s（但 burst 内 10.24 μs） |
| 不模糊范围（相位） | ±50 Hz       | ±50 Hz   | ±50 Hz                         | **±48.8 kHz**          |
| 不模糊范围（幅度） | —            | —        | ±32.5 kHz（M≤3）/ ±19.5 kHz（M≤5） | —                      |
| TCXO 最大差频 | —            | —        | **24.5 kHz**（5 ppm × 4.9 GHz）  | 同左                     |
| 能否覆盖最大差频  | 否            | 否        | M=2,3 可覆盖（32.5>24.5 kHz）✓      | 是 ✓                    |
| 能否测真实 CFO | 否            | 否        | 粗估（>3 kHz 有效）                  | 是                      |
| 大CFO下幅度损失 | ~19 dB（7kHz） | ~0.2 dB  | —                              | —                      |
| 测量精度      | 余数精度好        | 余数精度更好   | 数百 Hz                          | ~4 Hz（5 min）           |
| 实现难度      | 现有硬件         | 只改参数     | 只改参数                           | 新软件                    |

## 5. 综合建议

1. **立即可做**：将 10ms 系统的平均数从 15 改为 2 或 3，重新录制 B2B 数据。此操作：
   - 提升相位测量 SNR，余数 CFO 估计更干净；
   - 不改变不模糊范围，仍需 [[B2B频偏估计与OTA校准方案]] 做校准；

2. **粗估真实 CFO**：用 $M = 2, 3, 4, 5$ 顺序录制 B2B，对幅度比做拟合。若结果显示 $f_{CFO} \gtrsim 3\ \text{kHz}$，说明幅度损失是实测动态范围受限的主要原因；

3. **精确测量真实 CFO**：用新 1s 软件（burst 内 15 个 ZC 不平均），这是目前唯一能可靠测出真实 ppm 级 CFO 的方法；

4. **最终校准链路**：用新 1s 软件测得 $\hat{f}_{CFO}$ → 代入 [[B2B频偏估计与OTA校准方案]] Step 2 对 10ms OTA 数据做逐帧相位补偿 → 相干叠加。
