# CFO 测量数据提取 - 可执行规范文档

**版本** : 3.0  
**日期** : 2026-04-08  
**关键** : FPGA平均输出 (滑动窗口1024pts)，幅度法估计真实CFO  
**方法** : 从时序中选M个连续帧，对其功率做平均，计算功率比值

---

## 执行概览

FPGA输出**平均后的IQ数据** (滑动窗口1024采样点)，使用**幅度法**估计真实CFO。

M=2,3,5,13 指：**从帧时序中选择M个连续帧，对其功率做时间平均**

```
原始测量文件 (.bin) [FPGA平均输出]
    ↓
帧解析、IQ提取 → (n_frames, 1024) complex64
    ↓
Task 1: 生成CIR & 计算帧级功率时序
    ├─ LFM matched filter相关 → CIR (n_frames, 1024)
    ├─ 主径bin定位
    └─ 计算帧级功率 P[n] (n_frames,)
    ↓
Task 2: 对M=2,3,5,13分别计算帧功率窗口平均
    ├─ M=2: P̄^(2) = 滑动窗口2帧的功率平均
    ├─ M=3: P̄^(3) = 滑动窗口3帧的功率平均
    ├─ M=5: P̄^(5) = 滑动窗口5帧的功率平均
    └─ M=13: P̄^(13) = 滑动窗口13帧的功率平均
    ↓
Task 3: 幅度法估计真实 f_CFO
    ├─ 计算功率比值 ρ(M_i, M_j) (共6个比值)
    ├─ 最小二乘拟合 Δφ
    └─ 还原 f_CFO = Δφ / (2π·T_ZC)
    ↓
Task 4: 生成输出图表
    ├─ PNG 1: 频偏时序 (3频段 × 4M值)
    ├─ PNG 2-4: PDP热力图 (3频段)
    └─ PNG 5-7: 第1000帧PDP (3频段，4条曲线对比)
```

## 第2章 处理任务详解

### Task 1: 数据加载与CIR生成
CIR生成参考：[[bin_reader_compact]]的1，2，3.

#### 1.1 数据加载

**输入** : `.bin` 文件路径  
**过程** :
1. 使用 `LuoyangBinReader` 读取FPGA平均输出
2. 解析为 (n_frames, 1024) complex64

**输出** : `iq` 形状 `(n_frames, 1024)` complex64

**关键** : 这已是FPGA平均后的数据，不再做IQ平均

#### 1.2 CIR生成

**过程** :
1. 构建LFM matched filter (频域)
2. DC移除：`iq_dc = iq - mean(iq, axis=1)`
3. 对每一帧执行FFT滑动相关
4. 提取中心窗口

**输出** : `cir` 形状 `(n_frames, 1024)` complex64

#### 1.3 主径bin定位

**过程** :
1. 计算PDP均值：`pdp_avg = |cir|² 沿n平均`
2. 找幅度最大值：`peak_bin = argmax(pdp_avg)`

**输出** : `peak_bin` (int)

#### 1.4 帧级功率计算

**过程** :
1. 在主径bin处提取复数值：`peak[n] = cir[n, peak_bin]`
2. 计算功率：`P[n] = |peak[n]|²`

**输出** : `P` 形状 `(n_frames,)` float64 — 帧级功率时序

---

### Task 2: 多M窗口功率平均

对M ∈ {2, 3, 5, 13}，分别执行：

#### 2.1 计算方法

**公式** :
$$\bar{P}^{(M)} = \frac{1}{n_{\text{windows}}} \sum_{i=1}^{n_{\text{windows}}} \text{mean}(P[i:i+M])$$

其中：
- 窗口大小 = M帧
- 窗口滑动（每帧移动一个位置）
- 计算所有可能的M帧滑动窗口的功率平均
- 再对所有窗口取总平均

**例子** (M=2):
```
P[0:2] → mean → p_w1
P[1:3] → mean → p_w2
P[2:4] → mean → p_w3
...
P̄^(2) = mean([p_w1, p_w2, p_w3, ...])
```

#### 2.2 输出

对每个M值，输出标量：`P_bar_M` = 该M下所有窗口的平均功率

---

### Task 3: 幅度法估计真实CFO

根据幅度保留因子与CFO的关系。

#### 3.1 功率比值计算

**原理** : 不同M值的主径功率与M和CFO都有关系

$$\bar{P}^{(M)} = P_0 \cdot g^2(M, \Delta\phi)$$

其中 $g(M, \Delta\phi) = \frac{|\sin(M\pi f_{\text{CFO}} T_{\text{ZC}})|}{M|\sin(\pi f_{\text{CFO}} T_{\text{ZC}})|}$

**消去$P_0$**，计算功率比值 (共6个):
```
ρ(2,3)  = P̄^(3) / P̄^(2)
ρ(2,5)  = P̄^(5) / P̄^(2)
ρ(2,13) = P̄^(13) / P̄^(2)
ρ(3,5)  = P̄^(5) / P̄^(3)
ρ(3,13) = P̄^(13) / P̄^(3)
ρ(5,13) = P̄^(13) / P̄^(5)
```

#### 3.2 最小二乘拟合

**目标** : 求 $\Delta\phi \in [0, \pi]$ 使残差最小

$$\text{minimize} : \sum_{i<j} \left( \rho_{\text{theory}}(M_i, M_j, \Delta\phi) - \rho_{\text{measured}}(M_i, M_j) \right)^2$$

**方法** :
- 网格搜索：在 $\Delta\phi \in [0, \pi]$ 上搜索
- 或使用非线性优化 (scipy.optimize)

**输出** : `Delta_phi_hat` (float, rad)

#### 3.3 还原真实CFO

$$\hat{f}_{\text{CFO}} = \frac{\hat{\Delta\phi}}{2\pi T_{\text{ZC}}}$$

其中 $T_{\text{ZC}} = 1024 / (50 \text{ MHz}) = 20.48$ μs

**输出** : `f_CFO_true` (float, Hz)

## 第3章 数据流向总结

```
.bin文件 (FPGA平均输出)
    ↓
1. 加载 → (n_frames, 1024) complex64
    ↓
2. CIR生成 + 主径功率 P[n]
    ↓
3. 分别对M=2,3,5,13做帧窗口平均
    ├─ M=2: P̄^(2)
    ├─ M=3: P̄^(3)
    ├─ M=5: P̄^(5)
    └─ M=13: P̄^(13)
    ↓
4. 计算6个功率比值 ρ(M_i, M_j)
    ↓
5. 最小二乘拟合 → Δφ
    ↓
6. 还原 f_CFO_true = Δφ / (2π·T_ZC)
    ↓
7. 生成7张PNG输出
```
